# Getting Started with calculator-app
this website renders one page and the page have two sides split vertically down the middle; the math problem on the left and the calculator on the right. There should be an ability to submit an answer and to skip the question. On submission of the question we want the screen to indicate if it was correct or incorrect. 


## To Start the project

first install:
### `npm install`
to download all the dependencies for this project.

then run:
### `npm start`

this runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.\
You will also see any lint errors in the console.
